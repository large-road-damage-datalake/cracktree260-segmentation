# CrackTree260

## Overview

CrackTree260 is a task-specific segmentation package derived from the DeepCrack dataset release.

- Task: segmentation
- Images: 260
- Annotations: 1272
- Classes: 1 (crack)
- Annotation format: paired bitmap masks

## Source Dataset

- Name: CrackTree260
- Source URL: https://github.com/qinnzou/DeepCrack
- DeepCrack paper: https://ieeexplore.ieee.org/document/8517148
